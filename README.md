## Building on Windows

Clone the repository with the `--recursive` flag:

```
git clone --recursive https://github.com/CriminalRETeam/gta2_re.git
```


Build instructions.

Windows 

```
python setup.py
build.bat
```

Linux

```
python setup.py
wine build.bat
```
